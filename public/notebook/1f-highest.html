Math.max()

Write a function called **max,** which accepts an array and returns the highest value.

Do not use the built-in `Math.max()` function!

Examples:

  • max([5, 1, 4, 7, 1, 2]); // 7
  • max([3, 4, 12, 1, 8]); // 12
  • max([-1, 6, 3, 2.2, -10, -4]); // 6

Algorithm

  • have a container for the highest value initialized to the lowest possible value in case of negative numbers
    • start the loop
  • if the item at index is larger than highest, assign to highest the content of item
  • finish loop
  • return the content of highest variable

twoHighest()

Write a function called **twoHighest** that takes an array of numbers as its argument and returns the **two highest numbers within the array**.

The returned value should be an array in the following format: `[secondHighest, highest]`

The order of the numbers passed in could be any order.

**Do not use the build in sort() method - the tests will fail!**

Examples:
  • twoHighest([1, 2, 10, 8]); // [8, 10]
  • twoHighest([6, 1, 9, 10, 4]); // [9,10]
  • twoHighest([4, 25, 3, 20, 19, 5]); // [20,25]
  • twoHighest([1, 2, 2]) // [2, 2];

Algorithm

  • have containers for both highest and secondHighest values initialized to the lowest possible value in case of negative numbers
    • start the loop
  • if the item at index is larger than highest, assign to highest the content of item, swap values
  • if the item at index is not larger than highest, check if item is larger than secondHighest, swap values
  • if item at index is not larger than highest, nor larger than secondHighest, do nothing
  • finish loop
  • return an array with secondHighest as first element, and highest as a second element
